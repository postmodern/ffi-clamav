## Description

Ruby [FFI][ffi] bindings for libclamav, part of [ClamAV], an anti-virus
toolkit for UNIX.

## Features

## Examples

    require 'ffi/clamav'

## Requirements

* [ffi] ~> 1.0

## Install

    $ gem install ffi-clamav
